 **Project 1: Rule-Based Chatbot**
__________________________________________________________________________________________________________________________________________
Rule-Based Chatbot 
Project Overview
This project is a simple rule-based chatbot built using Python and ipywidgets .
The chatbot responds to predefined user inputs such as greetings, time, date, jokes, and help commands. It uses conditional logic to analyze user text and generate appropriate responses.
This project demonstrates the fundamentals of chatbot design without using machine learning.
________________________________________
Features
•	 Interactive UI using ipywidgets
•	 Rule-based text processing
•	 Greeting detection (hello, hi, hey)
•	 Time and date response
•	 Joke generator
•	 Basic help menu
•	 Exit/disable chat functionality
•	 Chat history display
________________________________________
 Technologies Used
•	Python 3
•	ipywidgets
•	datetime module
________________________________________
 How It Works
1.	The user types a message.
2.	The input is converted to lowercase.
3.	The program checks for keywords using conditional statements.
4.	If a match is found, a predefined response is returned.
5.	If no match is found, a default fallback message is shown.
This chatbot follows a rule-based approach, meaning it does not learn from conversations — it responds based on fixed conditions..
________________________________________
Supported Commands
You can try:
•	hello
•	how are you
•	your name
•	time
•	date
•	joke
•	weather
•	help
•	bye
Typing bye or exit will disable the chat input.

**Project 2: Tic Tac Toe with Minimax AI**
__________________________________________________________________________________________________________________________________________
Project Overview
This is a command-line Tic Tac Toe game built using Python.
The player competes against an AI opponent powered by the Minimax algorithm, making it unbeatable.
The game runs entirely in a standard Python environment (no external libraries required).
________________________________________
 Features
•	 Human vs AI gameplay
•	 Unbeatable AI using Minimax algorithm
•	 Draw detection
•	 Winner detection
•	 Input validation
•	 Replay option
________________________________________
How the AI Works
The AI uses the Minimax algorithm, a recursive decision-making algorithm commonly used in turn-based games.
•	The AI (O) tries to maximize the score.
•	The Player (X) tries to minimize the score.
•	The algorithm evaluates all possible future moves.
•	The AI always chooses the optimal move.
Because of this, the AI:
•	Never loses
•	Always forces a win or draw
________________________________________
Game Controls
The board positions are numbered as:
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
•	Enter a number (1–9) to place X
•	The AI automatically places O
•	The game ends when there is a winner or a draw
